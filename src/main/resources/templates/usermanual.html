User Manual

Hi. This is the User Manual of Boxing Club Staff Rostering Management System.
1.The index page is `http://localhost:8080/index`. Click `sign in` to sign in the system.
2.Click `management` to go to the management page.

For admin:
1. user management: You can add, edit, delete, search users in the system. `type` means `job title`.
2. set type: Type can be seen as `job title` or `department`. There are 6 types defined in the system. You can add, edit or delete types.
3. set shift group: A shift group may have several shifts. For example, there are shifts for boxers in workdays and shifts for boxers in weekends. You can create a shift group named `boxer` and every shift belongs to this group. Or you can create two shift group `boxer weekdays` and `boxer weekends`. Shift groups are defined for scheduling work shift. So a shift group should be associated with a type(for instance, boxers must join boxers' shift group).
4. set shift: You can add, edit, delete, search shifts in the system.
5. set labor law: If your club obeys the labor law, you should set the longest labor time per week. If the scheduling work shift breaks the law, the system will warn.
6. begin to schedule: First, set the start date and end date. You can add excluded date as well. Choose the shift group using in scheduling. Then, choose dates applied to each shift. Click `submit` to get the scheduling work shift. You can switch from shift view to people view. Besides, it shows labor statistics. If you want to save it, input the file name then click `save work shift`.
7. view work shift: You can view the saved work shifts. Click `edit` icon to view it. You can edit table by clicking `edit table`. You can add, change or delete the employee of the shift. Finally, you can override the work shift or save as the new work shift.
8. change shift: You can see the list of applications of changing shift. If A wants to change shift with B, A is `applicant` and B is `exchange employee`. If the exchange employee has agreed the application, you can click edit button to decide whether to approve the application or refuse it. Then, do not forget to edit the work shift.

For employee:
1. user management: You can change your password.
2. view work shift: You can view the work shift by clicking the icon.
3. apply for changing shift: If you want to change the shift, you should first consult with someone. Then click `+` to create a new application. After submission, you can see the status in the list. If it is approved, it means that the manager has approved your application.
4. agree to change shift: If your colleague consulted with you for changing shift, he or she will create an application in the system. You need to agree it. ,
